# Data Preprocessing Assignment

##  Overview

This project demonstrates how to clean and prepare raw customer data for machine learning. The dataset contains missing values, duplicate records, and categorical features that need to be processed before model training.

---

##  Task 1 — Data Cleaning

* Filled missing values in **age** using median
* Filled missing values in **city** using mode
* Removed duplicate records

---

##  Task 2 — Encoding

* Applied **One-Hot Encoding** to the city column
* Applied **Label Encoding** to the gender column

---

## Task 3 — Feature Scaling

* Used **Min-Max Scaling** to normalize values between 0 and 1
* Used **Standardization** to scale data with mean = 0 and standard deviation = 1
